Calculates the distance between this GeoVector2F and another GeoVector2F.

The calculation used to obtain the distance can be seen in this example:

\begin{align*} \vec{v}_{1} &= \begin{pmatrix} 1\\ 2 \end{pmatrix}\\ \vec{v}_{2} &= \begin{pmatrix} 6\\ 5 \end{pmatrix}\\ \vec{\Delta v} &= \vec{v}_{2} - \vec{v}_{1}\\ &=\begin{pmatrix} 6 - 1\\ 5 - 2 \end{pmatrix} \\ &= \begin{pmatrix} 5\\ 3 \end{pmatrix}\\ \left \| \vec{\Delta v} \right \| &= \sqrt{{\Delta v}_x^2 + {\Delta v}_y^2}\\ &= \sqrt{5^2 + 3^2} \rightarrow \sqrt{25 + 9}\\ &= \sqrt{36}\\ &= 6 \end{align*}

This is a pure method. Calling this without using the result has no effect and can introduce overhead for calling the method.

GeoVector2F NovelRT::Maths::GeoVector2F::GetNormalised ( ) const
inlinenoexcept

Constructs a unit GeoVector2F with a magnitude of one by normalising this vector.

To normalise a vector you need to divide its components by its magnitude. As a result, a vector with the resulting components will have a magnitude of 1, also known as a unit vector.

\begin{align*} \vec{v}&= \begin{pmatrix} 4\\ 3 \end{pmatrix}\\ \left \| \vec{v} \right \| &= \sqrt{v_{x}^{2} + v_{y}^{2}} = \sqrt{4^{2} + 3^{2}} \rightarrow \sqrt{16 + 9} = \sqrt{25} = 5\\ \hat{v}&=\frac{\vec{v}}{\left \| \vec{v} \right \|} =\frac{\bigl(\begin{smallmatrix} 4\\ 3 \end{smallmatrix}\bigr)}{5} = \begin{pmatrix} \tfrac{4}{5}\\ \tfrac{3}{5} \end{pmatrix}\\ \left \| \hat{v} \right \| &= \sqrt{\tfrac{4}{5}^{2} + \tfrac{3}{5}^{2}} = \sqrt{\tfrac{16}{25} + \tfrac{9}{25}} \rightarrow \sqrt{\tfrac{25}{25}} \rightarrow \sqrt{1} = 1 \end{align*}

This is a pure method. Calling this without using the result has no effect and can introduce overhead for calling the method.

Returns

\begin{align*} \begin{pmatrix} 8\\ 6 \end{pmatrix} \oslash \begin{pmatrix} 2\\ 3 \end{pmatrix} &= \begin{pmatrix} \tfrac{8}{2}\\ \tfrac{6}{3} \end{pmatrix}\\ &= \begin{pmatrix} 4\\ 2 \end{pmatrix} \end{align*}

float NovelRT::Maths::GeoVector2F::SquaredDistance ( GeoVector2F  other)
inlinenoexcept

Calculates the squared distance between this GeoVector2F and another GeoVector2F.

The calculation used to obtain the squared distance can be seen in this example:

\begin{align*} \vec{v}_{1} &= \begin{pmatrix} 1\\ 2 \end{pmatrix}\\ \vec{v}_{2} &= \begin{pmatrix} 6\\ 5 \end{pmatrix}\\ \vec{\Delta v} &= \vec{v}_{2} - \vec{v}_{1}\\ &=\begin{pmatrix} 6 - 1\\ 5 - 2 \end{pmatrix} \\ &= \begin{pmatrix} 5\\ 3 \end{pmatrix}\\ \left \| \vec{\Delta v} \right \|^2 &= {\Delta v}_x^2 + {\Delta v}_y^2\\ &= 5^2 + 3^2 \rightarrow 25 + 9\\ &= 36 \end{align*}

This is a pure method. Calling this without using the result has no effect and can introduce overhead for calling the method.
