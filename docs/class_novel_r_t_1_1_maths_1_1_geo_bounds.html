◆ IntersectsWith()

bool NovelRT::Maths::GeoBounds::IntersectsWith ( GeoBounds  otherBounds) const

Evaluates whether another GeoBounds instance intersects with this GeoBounds instance or not.

This implementation does not support rotation, so only AABB intersection computation is allowed.

This is a pure method. Calling this without using the result has no effect and can introduce overhead for calling the method.

Returns
true if the otherBounds intersects with this bounding box, otherwise false.
Exceptions
Exceptions::NotSupportedExceptionIf either bounding box has a rotation set to a value other than 0.
